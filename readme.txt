Plugin para crear textos legales y cumplir RGPD fácil y rápido rellenando un formulario con los datos del negocio.

== Description ==

El plugin Avisos Legales RGPD permite a los administradores de sitios web crear fácilmente las páginas legales necesarias para cumplir con el Reglamento General de Protección de Datos (RGPD) y otras normativas legales. Con un simple formulario, los usuarios pueden generar:

* Política de Privacidad
* Aviso Legal
* Política de Cookies
* Declaración de Accesibilidad

== Installation ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/avisos-legales-rgpd`, o instala el plugin directamente desde el panel de WordPress.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Ve a 'Avisos Legales' en el menú de administración y completa el formulario con la información de tu sitio web.

== Frequently Asked Questions ==

= ¿Este plugin me hace cumplir automáticamente con el RGPD? =

Este plugin te ayuda a crear las páginas legales necesarias, pero el cumplimiento del RGPD implica más que solo tener estas páginas. Asegúrate de implementar las medidas técnicas y organizativas adecuadas para cumplir plenamente con el RGPD.

= ¿Puedo personalizar los textos generados? =

Sí, una vez que las páginas son creadas, puedes editarlas como cualquier otra página de WordPress para ajustar el contenido a tus necesidades específicas.
